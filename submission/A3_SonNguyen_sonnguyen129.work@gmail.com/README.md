## 5. Data Disclosure

- **Provided data:** `train.json` (2,986 samples), `dev/validation` (584 samples), `test.json` (494 samples) from the course/Kaggle competition. No other datasets were used.
- **Synthetic data:** few-shot CoT reasoning traces generated by Qwen3.5-4B itself during the Phase 2 evolution loop; each trace was kept only if its DSL program executed to the gold answer (execution-verified). Used as few-shot examples in the final prompts.
- **External data:** none. No external datasets, no external labels, no test-set reconstruction of any kind.
- **Models:** `QuantTrio/Qwen3.5-4B-AWQ` (self-hosted, vLLM on Cerebrium A10). DeepSeek-R1-Distill-Qwen-7B was tested as a judge in an ablation and **rejected** (not part of the final system). No external paid APIs.
- **Compute:** ~14–15 GPU-hours A10 on Cerebrium; ~64.9M input / ~17.5M output tokens total.
